# 🚜 Construction & Mining Equipment AI Pipeline
Developed by: **Eng. Raafat Hazza**

## 🎯 Project Overview
An automated AI system designed for real-time monitoring of heavy machinery productivity and site safety. Utilizing YOLOv8, the system tracks equipment (Excavators & Trucks) and calculates work cycles.

## 🚀 Key Features
- **Object Detection:** High-accuracy recognition of Liebherr & CAT equipment.
- **Safety Geofencing:** Real-time danger zone alerts to prevent accidents.
- **Productivity Analytics:** Automated logging of cycle times and tonnage.
- **Containerized:** Ready for deployment using Docker.

## 🛠️ Tech Stack
- **AI:** YOLOv8 (Ultralytics)
- **Vision:** OpenCV
- **Backend:** Python
- **DevOps:** Docker & Docker Compose
